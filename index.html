wkhtmltopdf-reset

A css file and a javascript file that you can use to ease the printing process

How to use

  • Add the jquery and wkhtmltopdf.js scripts
  • Add the wkhtmltopdf.css stylesheet
  • Set the class on the body to something like class="a4 landscape"

What do you get?

Some helpers

  • class="wk-page" a segment that will fill one page (like this page)
  • class="wk-table" a table that (using js) breaks up per-page, a copy of <tr class="header"> is repeated on each page
  • class="wk-html" a segment denoting general HTMLness
  • class="wk-hiddentitle" applied to h1, h2, h3 will be hidden from view, but still count toward the TOC and headers / footers pages
  • You can also load data by passing a cookie (like this page)

Nota Bene

  • You must use the --disable-smart-shrinking flag to make this work
  • Tested with the 11.0 static binary

./wkhtmltopdf-i386 --page-size A4 --orientation landscape --disable-smart-shrinking --dpi 150 --header-html header.html --footer-html footer.html index.html index.html.pdf

You can load data like so

./wkhtmltopdf-i386 --page-size A4 --orientation landscape --disable-smart-shrinking --cookie datafile './data.js' --dpi 150 --header-html header.html --footer-html footer.html index.html index.html.pdf

Then, in your javascript use: wk.loaddatafile(success, error); where success and error are regular jQuery functions
